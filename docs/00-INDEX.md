## What is Ranex?

Ranex is an **AI-native governance framework** that makes AI-generated code production-ready. It solves three critical problems:

| Problem | Solution | How Ranex Helps |
|---------|----------|-----------------|
| **Logic Drift** | State Machine | Enforces business rules at runtime |
| **Hallucination** | Import Validation | Detects typosquatted packages |
| **Security Issues** | SAST Scanner | Catches vulnerabilities in AI code |
